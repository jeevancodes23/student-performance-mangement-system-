Project Description
Student Performance Management System is a Python-based application developed to manage student academic records. This project helps users to add, update, delete, search, and view student performance details.
features
- Add Student Details
- Update Student Records
- Delete Student Records
- Search Student Details
- View Student Performance

Technologies Used
- Python
- SQLite / MySQL
- Tkinter (GUI)
How to Run
1. Clone this repository.
2. Install Python.
3. Run the project using:
   python app.py
